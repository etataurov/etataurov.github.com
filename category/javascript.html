Ember Foundation Modal

Sat 15 February 2014
By Evgeny Tataurov

In Javascript.

Overview

For my new idea I needed a pretty smart client-side app, so i decided to try some JavaScript MVC framework. I chosed EmberJS, without much research, just because i heard of it, and it seemed powerful and easy. And for css framework i chosed Foundation, because I already tried Bootstrap and wanted something new.

Problem

My application needed many modal windows(I am not sure if it is bad or good practice, but I wanted them). I found in Foundation documentation about modal windows, it was really easy to implement them, but i wanted to integrate them with Ember's routes, views and models.

Existing solutions
